# CETMS

### 环境说明

1. IDEA
2. JDK 1.8
3. MySQL 5.7.42
4. MySQL-connector-8.0.33

### 账户密码

- 默认账户: admin
- 默认密码 :admin

### 软件架构

- 2 层 C/S 架构
- MVC 模式

### 软件功能

- 多用户：管理员、教师、学生
- 管理员：管理教师和学生信息
- 教师：管理大学英语四六级成绩
- 学生：查询大学英语四六级成绩
